# image-processor-pro

### 1. Install `uv`

If you don't have it, install it once globally:

* **Windows:** `powershell -c "irm https://astral.sh/uv/install.ps1 | iex"`
* **Linux/Mac:** `curl -sSf https://astral.sh/uv/install.sh | sh`

### 2. Clone and Sync

Once you've pulled the repository, navigate to the folder and run:

```bash
uv sync

```
### 3. Running the App

```bash
uv run src/main.py

```
